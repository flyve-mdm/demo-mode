What is it?

This plugin for GLPI is a subproject of Flyve MDM.

The demo mode enables the self-creation accounts from the Web Admin Dashboard, allowing the registration of new users to the GLPI interface, without requesting access.


Overview

Demo Mode plugin

The demo plugin works with functions of Flyve MDM plugin for GLPI, therefore it is required its previous installation, check the following matrix to know which versions are compatible:

Compatibility matrix

GLPI 9.1.2 9.1.3 9.1.4 9.2.0
Flyve MDM 1.x.x 1.x.x 1.x.x 2.0.0-dev
Demo Mode - - - 1.0.0-dev

Features

Registration

The Demo Mode provides the self - registration of new users to GLPI, simultaneously allowing access to the Flyve MDM features from both the web console and the interface integrated in GLPI.

Time trial

Limit the use of Flyve MDM to new registered users after a period of time.
